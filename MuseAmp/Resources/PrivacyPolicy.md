# Privacy Policy

Last updated: April 15, 2026

## Overview

Muse Amp collects no personal data and uploads no personal data to Muse Amp servers.

## On-Device Data

Muse Amp stores the data it needs on your device, including local audio files, artwork and lyrics cache, playlists, playback state, diagnostic logs, and the server settings you enter.

## Your Music Server

If you connect a Subsonic-compatible server, your device connects directly to the server address you configure. Muse Amp uses your saved server credentials only for that connection.

## Local Network Transfer

If you transfer songs between devices, the transfer happens directly on your local network. Muse Amp uses no relay server for this feature.

## Camera

Camera access is used only for QR code scanning during transfer setup. Camera data stays on device.
